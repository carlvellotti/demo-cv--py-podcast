# Product Context

## What We're Building

**Moltbot** - An AI-powered notification management system that learns user preferences and reduces notification fatigue.

## Current Version (v1)

- Basic notification filtering by app/channel
- Manual quiet hours
- Simple digest emails (daily summary)
- ~2,000 active users

## v2 Focus (Current)

**Smart Notifications** - ML-powered priority scoring
- Learns from user behavior (what they click, dismiss, snooze)
- Predicts notification importance
- Auto-categorizes into urgent/normal/low
- Customizable notification settings UI

## Key Metrics

| Metric | Current | Target |
|--------|---------|--------|
| Daily Active Users | 2,000 | 5,000 |
| Notification Click Rate | 12% | 25% |
| User-reported "overwhelm" | 45% | <20% |
| Digest Open Rate | 34% | 50% |

## Competitors

- **Slack Focus Mode** - Basic DND, no learning
- **Apple Focus** - System-level, app-based rules
- **Quiet for Gmail** - Email only, no cross-app
