// How It Works

Three steps to launch

From zero to marketplace in under two weeks. Our tooling handles the hard parts.

01

Register

Create your developer account and get your API credentials. Access sandbox environments instantly.

$ moltbot init my-skill
# Creates skill scaffold
02

Build

Use our SDKs and CLI tools to develop locally. Hot-reload in dev mode. Test across all platforms.

$ moltbot dev --watch
# Live reload enabled
03

Publish

Submit for review. Once approved, your skill goes live on ClawdHub for the entire community.

$ moltbot publish
# Submitted for review
