# Temp

Scratch space for one-off tasks.

**Anything in this folder can be safely deleted.**

## What Goes Here

- Files you want Claude to analyze
- Screenshots for context
- Exports from other tools
- Drafts you're working on

## Example Uses

- Drop a PDF: "Summarize this document"
- Drop a screenshot: "What's wrong with this UI?"
- Drop a CSV: "Analyze this data"

## Cleanup

Periodically delete everything in here to keep things tidy.
