# Tools

Scripts and utilities that Claude can run.

## What Goes Here

- Python scripts for data processing
- Shell scripts for automation
- Any executable that Claude might need to run

## Examples

- `export-linear-issues.py` - Export Linear issues to CSV
- `summarize-meetings.py` - Summarize all meetings from the past week
- `format-prd.py` - Convert PRD to different formats

## Usage

Claude can run these directly:

```
Run the export-linear-issues script
```

Or you can run them manually:

```bash
python tools/export-linear-issues.py
```
